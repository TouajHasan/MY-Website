Directory Brute-Forcing Tool User Manual

Features

  • Multithreading: Perform concurrent requests to speed up the brute-forcing process.
  • Customizable Status Codes: Specify which HTTP status codes to match.
  • Progress Saving: Save the progress of completed words to resume later.
  • Output Logging: Save discovered URLs and log the activity.
  • URL Validation: Check if the provided URL is in the correct format.
  • Case Transformation: Transform words to uppercase, lowercase, or capitalized form.

Introduction

This tool is designed for directory brute-forcing, allowing users to discover hidden paths on a web server. It uses multithreading to enhance performance and supports various options for customization.

Requirements

  • Python 3.x
  • Requests library
  • Access to a terminal or command prompt

Installation

  1. Ensure Python 3.x is installed on your system.
  2. Install the Requests library if it's not already installed:
    pip install requests

Usage

Run the tool using the following command format:

python brute_force.py -u <TARGET_URL> -w <WORDLIST_PATH> [OPTIONS]

Arguments

  • -u, --url (required): The target URL where 'THS' indicates the position to brute-force.
  • -w, --wordlist (required): Path to the wordlist file containing potential directories/files.
  • -t, --threads (optional): Number of threads to use (default: 30).
  • -mc, --match-codes (optional): Comma-separated list of HTTP status codes to match (default: 200,204,301,302,307,403,500,502,503).
  • --log (optional): Path to a log file where the activity will be recorded.
  • -r, --resume (optional): Resume scanning from the last completed word.
  • --suffixes (optional): Add custom suffixes to all wordlist entries (comma-separated).
  • -U, --uppercase (optional): Convert wordlist entries to uppercase.
  • -L, --lowercase (optional): Convert wordlist entries to lowercase.
  • -C, --capital (optional): Capitalize the first letter of each word in the wordlist.

Example Commands

  1. Basic Usage: python brute_force.py -u http://example.com/THS -w wordlist.txt
  2. Using Custom Status Codes: python brute_force.py -u http://example.com/THS -w wordlist.txt -mc 200,403
  3. Using Multithreading: python brute_force.py -u http://example.com/THS -w wordlist.txt -t 50
  4. Resume Previous Scan: python brute_force.py -u http://example.com/THS -w wordlist.txt -r

Output

The tool will display results in the terminal and save found URLs to the specified log file if the --log option is provided. The status codes matched will also be displayed in the output.
